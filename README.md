# A/B-тест: новая страница и старая (Python, статистика)

Анализ A/B-теста интернет-магазина: **статистическая проверка гипотез** на Python.
Вопрос - реально ли новая версия страницы лучше, или разница случайна?

![Результаты A/B-теста](ab_result.png)

## Задача
Тестируем новую версию страницы (**B**) против старой (**A**). Нужно проверить
**статистически**, повышает ли B конверсию и выручку, или наблюдаемая разница -
просто случайность выборки.

## Данные
`ab_test.csv` - 10 000 пользователей: `group` (A/B), `converted` (0/1), `revenue`.

## Метод
- **Конверсия** (доля) --> **z-тест для двух долей** + 95% доверительный интервал.
- **ARPU** (выручка на пользователя, число) --> **t-тест Уэлча**.
- Порог значимости **α = 0.05**.

## Результаты
| Метрика | A (старая) | B (новая) | Тест | p-value | Значимо? |
|---|---|---|---|---|---|
| Конверсия | 9.35% | 12.03% | z-тест долей | 0.000014 | Да |
| ARPU | 4.62 | 6.16 | t-тест Уэлча | 0.000002 | Да |

- Прирост конверсии: **+2.68 п.п.** (95% ДИ [1.47; 3.89]) ≈ **+29%** относительно.
- Обе метрики значимо выше у новой страницы.

## Вывод и рекомендация
Новая страница **B статистически значимо** повышает и конверсию, и выручку на
пользователя (p < 0.001 для обеих метрик). **Рекомендация: выкатить B на всех
пользователей.**
